## Overview
This project analyzes customer churn behavior in a streaming platform.

The objective is to identify churn drivers, segment users,
predict churn risk, and generate business recommendations
through interactive Power BI dashboards.
Identify the key drivers behind customer churn
Segment customers into meaningful behavioral groups
Build a predictive model to flag at-risk customers
Develop a business KPI framework for ongoing monitoring
Deliver an interactive Power BI dashboard for stakeholders
Translate analysis into clear, actionable retention recommendations

## Key Insights

• Overall churn exceeded 50%

• Regional differences significantly impact churn

• Payment methods influence retention

• Tenure strongly affects churn probability

• Subscription plans show different churn behavior
